# React Hooks Playground

A comprehensive, interactive learning platform for mastering React Hooks. This project demonstrates practical implementations of all major React Hooks through hands-on examples, making it perfect for developers looking to deepen their understanding of state management and side effects in React applications.

## 🚀 Features

- **Interactive Demos**: Six dedicated practice areas for each major React Hook
- **Real-world Scenarios**: Practical examples including todo lists, API fetching, shopping carts, and more
- **Performance Insights**: Visual demonstrations of optimization techniques with useMemo and useCallback
- **Accessibility First**: Built with screen reader support and keyboard navigation
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
- **Modern Stack**: Latest React 19 with Vite, Tailwind CSS, and React Router

## 📚 What's Included

### Core Hook Demonstrations
- **useState**: Managing local component state with counters, forms, and interactive UI
- **useEffect**: Handling side effects, API calls, and cleanup operations
- **useMemo**: Performance optimization through memoized calculations
- **useCallback**: Preventing unnecessary re-renders with stable function references
- **useContext**: Global state management without prop drilling
- **useReducer**: Complex state logic with reducer patterns

### Practical Examples
- Todo list management with CRUD operations
- Real-time search filtering with performance comparisons
- E-commerce cart simulation with inventory tracking
- API data fetching with error handling and loading states
- User profile management with context providers

## 🛠️ Tech Stack

- **Frontend**: React 19, React Router 7
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4, Custom design system
- **Icons**: Heroicons (SVG-based)
- **API**: JSONPlaceholder and custom mock APIs
- **Development**: ESLint, Hot Module Replacement

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ and npm
- Modern web browser with JavaScript enabled

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/react-hooks-playground.git
cd react-hooks-playground
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory, ready for deployment.

## 🎯 How to Use

1. **Navigate the Menu**: Start from the main menu to explore different hook demonstrations
2. **Interact with Examples**: Each demo includes interactive elements to see hooks in action
3. **Check the Console**: Open browser dev tools (F12) to see performance logs and state changes
4. **Compare Approaches**: Switch between optimized and non-optimized versions to understand performance differences
